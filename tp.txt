git est un logiciel de versioning qui permet de versionner votre code
c est a dire gerer les versions de votre code au fur et a mesure que vous le modifiez

git appartient au modele de logiciels de gestion de versions distibue ou toutes les machines ont acces
a la base de code et pas besoin de passer par un serveur central

Activer dossier comme repository
-------------------------------
pour activer un dossier comme repository git , il faut   qu on se place dans ce dossier au niveau du terminal
la commande est : git init
